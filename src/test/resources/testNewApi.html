ChangeLog

What's New

  1. GET/accounts/{regNo}-{accountNo} gets the information from a single account
  2. PUT/accounts/{regNo}-{accountNo} Create new or update existing account
  3. GET/accounts lists accounts
  4. GET/account-events-metadata metadata for the events endpoint
  5. GET/account-events obtain all events emitted by the account-event service
  6. GET/account-events/{category} obtain all events scoped to a certain category
  7. GET/account-events/{category}/{id} obtain the individual events from an account
  8. GET/accounts/{regNo}-{accountNo}/reconciled-transactions/{id} obtain a single reconciled transaction from a given account
  9. PUT/accounts/{regNo}-{accountNo}/reconciled-transactions/{id} Create new or update reconciled transaction
  10. GET/accounts/{regNo}-{accountNo}/reconciled-transactions obtain reconciled transactions (added API capabilities not though not implemented)
  11. GET/accounts/{regNo}-{accountNo}/transactions/{id} obtain the individual single transaction from an account
  12. PUT/accounts/{regNo}-{accountNo}/transactions/{id} creates a single transaction on an account
  13. GET/accounts/{regNo}-{accountNo}/transactions obtain all transactions on account for a given account
